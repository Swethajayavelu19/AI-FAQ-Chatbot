 🤖 AI FAQ Chatbot

 📌 Overview
AI FAQ Chatbot is a web-based chatbot developed using Python and Flask. It answers frequently asked questions by applying Natural Language Processing (NLP) techniques with TF-IDF Vectorization and Cosine Similarity to find the most relevant response.

 🚀 Features
- AI-powered FAQ chatbot
- Natural Language Processing (NLP)
- TF-IDF Vectorization
- Cosine Similarity matching
- Professional web interface
- Responsive design
- Chat history
- Confidence score for responses
- Simple and easy to use

 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

 📂 Project Structure

AI-FAQ-Chatbot/

 ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/AI-FAQ-Chatbot.git
```

2. Install the required packages:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

4. Open your browser and visit:
```
http://127.0.0.1:5000
```

 📷 Output
The chatbot provides answers to frequently asked questions using NLP techniques and displays the response with a confidence score through a web interface.
